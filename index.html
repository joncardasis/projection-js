// projection.js

A minimal 3D wireframe renderer built from first principles using vanilla JavaScript and HTML Canvas 2D.


This project intentionally avoids WebGL, Three.js, and external dependencies as a learning exercise to expose the core mathematics behind 3D graphics including coordinate systems, projection, transformations, and camera placement by implementing a 3D rendering pipeline on a 2D canvas.

  • Model loading: .obj files are parsed into vertex/edge arrays
  • Projection: x/z, y/z perspective division
  • Rotation: 2x2 matrix transform in the xz-plane
  • Auto-fit camera: The camera is positioned to keep all model coordinates within the [-1,1] normalized device coordinate (NDC) space, accounting for the bounding sphere radius
