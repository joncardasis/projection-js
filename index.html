// projection.js

A minimal 3D wireframe renderer built from scratch with zero dependencies. No WebGL, no Three.js. Just Canvas 2D and vanilla JavaScript using perspective projection.

  • Model loading: Parse .obj files into vertex/edge arrays
  • Projection: Simple x/z, y/z perspective division
  • Rotation: Pure 2x2 matrix transform in the xz-plane
  • Auto-fit camera: Solve for z-distance keeping coordinates within [-1,1] NDC, accounting for bounding sphere radius
